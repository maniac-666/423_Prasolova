## Реализованный функционал
Приложение разделено на три логические страницы:
1. **Линейный алгоритм:** Расчет функции $w = |cosx - cosy|^{(1 + 2sin^2y)} \cdot (1 + z + \frac{z^2}{2} + \frac{z^3}{3} + \frac{z^4}{4})$.
2. **Разветвляющийся алгоритм:** Выбор функции $f(x)$ ($sh(x), x^2, e^x$) и расчет условий для $d$.
3. **Циклический алгоритм:** Табулирование функции $y = ax^3 + cos^2(x^3 - b)$ и построение динамического графика.

Описание тестов
TestMethod1 — Тренировочный тест
Тест демонстрирует работу основных методов класса Assert:
МетодОписаниеAssert.AreEqual(res, 4)Проверяет, что res равно 4Assert.AreNotEqual(res, 5)Проверяет, что res не равно 5Assert.IsFalse(res > 5)Проверяет, что выражение res > 5 — ложьAssert.IsTrue(res < 5)Проверяет, что выражение res < 5 — истина
TestFormula1_Linear — Тест линейного алгоритма
Проверяет вычисление функции w = |cos(x) - cos(y)|^(1 + 2sin²y) * (1 + z + z²/2 + z³/3 + z⁴/4) при различных наборах входных данных, включая граничные значения (x=0, y=0, z=0) и проверку поведения при z=0.
TestFormula2_Branching — Тест разветвляющегося алгоритма
Проверяет все три ветви условного оператора и все три варианта функции f(x):

Ветка x > y — вычисление d = (f(x) - y)³ + arctg(f(x))
Ветка y > x — вычисление d = (y - f(x))³ + arctg(f(x))
Ветка x = y — вычисление d = (y + f(x))³ + 0.5

TestFormula3_Cyclic — Тест циклического алгоритма
Проверяет вычисление функции y = ax³ + cos²(x³ - b) при различных значениях параметров a, x, b, включая проверку граничных случаев (a=0, отрицательный x) и свойства cos² ∈ [0, 1].
