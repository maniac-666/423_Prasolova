## Цель работы
Приобретение практических навыков разработки WPF-приложений с использованием навигации, математических вычислений и визуализации данных через диаграммы.

## Реализованный функционал
Приложение разделено на три логические страницы:
1. **Линейный алгоритм:** Расчет функции $w = |cosx - cosy|^{(1 + 2sin^2y)} \cdot (1 + z + \frac{z^2}{2} + \frac{z^3}{3} + \frac{z^4}{4})$.
2. **Разветвляющийся алгоритм:** Выбор функции $f(x)$ ($sh(x), x^2, e^x$) и расчет условий для $d$.
3. **Циклический алгоритм:** Табулирование функции $y = ax^3 + cos^2(x^3 - b)$ и построение динамического графика.

## Инструкция по запуску
1. Склонируйте репозиторий.
2. Откройте файл решения `.sln` в Visual Studio 2019/2022.
3. Убедитесь, что подключены ссылки на `System.Windows.Forms` и `WindowsFormsIntegration`.
4. Запустите проект (F5).
